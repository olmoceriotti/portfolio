Projects

A collection of some of the projects I worked on in the past years.

Heterogeneous Graph Neural Networks for Action Prediction on Egocentric Action Scene Graphs

In this project I worked on implementing heterogenous graph neural networks in PyTorch Geometric to work on EASG, a novel annotation frameworks for egocentric videos. The task tested was the one of action prediction. More on this work here.

PySteg

PySteg is a steganography suite written in Python. It performs steganography on different image formats with a wide variety of algorithms and QR Codes. More on the implementation and the usage here.

JSONParser

JSONParser is a collection of methods and function to parse JSON files into structures understandable by Prolog and LISP programs. Two different parsers are present, one written in prolog and on written in LISP. More on this project here.
